# Numerical Functions in SQL
## Round
```SQL
SELECT ROUND(AVG(Price), 2) FROM table1;
```

## Truncate
Rounds down the value
```SQL
SELECT TRUNCATE(AVG(Price), 0) FROM table1;
```

## Abs
Returns Absolute Value
```SQL
SELECT ABS(price) - ABS(length) FROM table1;
```
